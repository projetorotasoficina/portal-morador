# Análise do Layout do Site Oficial - Prefeitura de Pato Branco

Fonte: https://patobranco.pr.gov.br

## Cores Principais

### Header (Topo)
- **Fundo azul escuro**: `#1a2b52` (aproximado) - Azul marinho institucional
- **Faixa superior azul claro**: `#00a8e8` (aproximado) - Azul ciano

### Elementos
- **Títulos**: Azul ciano `#00a8e8`
- **Ícones de serviços**: Círculos azuis `#0891b2` (aproximado)
- **Texto principal**: Preto/cinza escuro
- **Fundo**: Branco e cinza claro

## Tipografia

- **Títulos principais**: Fonte sans-serif bold (provavelmente Montserrat ou similar)
- **Texto corpo**: Fonte sans-serif regular
- **Estilo**: Moderno, limpo, institucional

## Estrutura do Header

1. **Barra superior azul claro**:
   - Links: "156 - Fala Cidadão", "Portal da Transparência", "Carta de Serviços"
   - Ícones de redes sociais (Facebook, Instagram, YouTube, LinkedIn)
   - Seletor de idiomas

2. **Barra principal azul escuro**:
   - Logo da prefeitura (brasão + texto "PREFEITURA DE PATO BRANCO")
   - Menu horizontal: Início, O Município, Serviços On Line, Secretarias, Servidores, Licitações e Contratos, Contato
   - Campo de busca

## Componentes Principais

### Cards de Serviços
- **Formato**: Grid de cards quadrados
- **Ícones**: Círculos azuis com ícones brancos
- **Estilo**: Minimalista, ícones flat design
- **Hover**: Efeito de destaque

### Seção de Estatísticas
- **Layout**: 4 colunas
- **Dados**: Área total, População, Protocolo (com botão), IDH
- **Cores**: Azul ciano para números/destaques

### Banner Principal
- **Carrossel**: Imagens full-width
- **Overlay**: Texto sobre imagens
- **Navegação**: Setas laterais

## Footer

- Informações de contato
- Endereço da prefeitura
- Horário de atendimento
- Links para redes sociais
- Fundo escuro (azul marinho)

## Elementos de Identidade Visual

1. **Brasão da prefeitura**: Sempre visível no header
2. **Paleta azul**: Variações de azul (escuro, médio, claro)
3. **Ícones circulares**: Padrão para serviços
4. **Layout limpo**: Muito espaço em branco
5. **Responsivo**: Design adaptável

## Recomendações para Adaptação

### Para o Portal do Morador:

1. **Header**:
   - Usar mesma estrutura de duas barras (azul claro + azul escuro)
   - Incluir logo da prefeitura
   - Menu simplificado: Início, Meu Perfil, Sair

2. **Cores**:
   - Primária: `#00a8e8` (azul ciano)
   - Secundária: `#1a2b52` (azul marinho)
   - Destaque: `#0891b2` (azul médio)
   - Fundo: `#f8f9fa` (cinza muito claro)

3. **Tipografia**:
   - Usar fonte sans-serif moderna (Inter, Montserrat ou Roboto)
   - Títulos em bold
   - Hierarquia clara

4. **Componentes**:
   - Cards com bordas sutis
   - Botões com cantos arredondados
   - Ícones circulares para ações

5. **Footer**:
   - Fundo azul escuro
   - Informações de contato da Secretaria de Meio Ambiente
   - Links úteis

## Diferenças do Layout Atual

### O que precisa mudar:

1. ❌ **Cores atuais**: Verde → Azul
2. ❌ **Header**: Simples → Duas barras institucionais
3. ❌ **Logo**: Pequeno → Destaque no header
4. ❌ **Tipografia**: Padrão → Institucional
5. ❌ **Footer**: Ausente → Completo com informações

### O que manter:

1. ✅ Responsividade
2. ✅ Cards de informação
3. ✅ Formulários limpos
4. ✅ Estrutura de navegação simples
